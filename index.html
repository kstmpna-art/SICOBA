Guía de implementación — Netlify / Vercel
Paso 1 — Preparar Google Sheets
Crear un Google Sheets con 3 solapas: AERONAVES, BUQUES y CONFIG. Ir a Extensiones → Apps Script → pegar el código del archivo SAR_AppsScript_Backend.js → ejecutar setup() una sola vez para crear la estructura automáticamente.
Paso 2 — Publicar Apps Script
En Apps Script: Implementar → Nueva implementación → Tipo: Aplicación web → Ejecutar como: Yo → Acceso: Cualquier usuario → Implementar → Copiar la URL generada → pegarla en el campo de arriba.
Paso 3 — Publicar en Netlify (gratis)
Ir a netlify.com → registrarse gratis → Add new site → Deploy manually → arrastrar este archivo HTML → El sistema queda en línea en segundos con una URL del tipo sar-pna.netlify.app. Dominio propio opcional.
Paso 4 — Power BI Desktop
Obtener datos → Web → URL del Apps Script + ?action=getPowerBI. Los datos llegan como JSON. Clic en "Actualizar" para refrescar todos los dashboards con los datos más recientes del Sheets.
